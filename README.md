# Adminos Example Project

This is an example of using gem [adminos](https://github.com/Molinos/adminos).


## Dependencies
* rvm
* ruby 2.5.1
* rails 5.2
* postgresql 10.x

## Get started

```sh
bundle install
rails db:create db:create
rails db:create db:migrate
rails db:create db:setup
```

```sh
rails s
./bin/webpack-dev-server
```
